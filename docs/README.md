## Setup

```sh
pnpx sv create svelte-terminal
cd svelte-terminal
pnpm run dev
```

And now you can edit and see live changes!

## Building

To create a production version of your app:

```sh
pnpm run build
```

You can preview the production build with `pnpm run preview`.
